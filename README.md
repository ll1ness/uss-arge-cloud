# USS(Update-Upload System) ARGECloud for users <img src="https://img.shields.io/badge/Status-In_development-black">

This system use for just ll1ness's projects and not using for other users.

### About
In folders in this repository contain all update scripts and files, which must download to your device. 

### Hot it works
1. from app sending https request to folder, which contain update script and other.
2. Downloading script (e.g. update.php, update-script.js) and run in app (download to res:// protocol and running)
3. Script end work and update success!
